# 大一全栈实战项目合集
本人大一期间完成的机器学习、Python编程、Web全栈开发项目，涵盖算法建模、爬虫自动化、前后端开发等方向。


## 核心技能模块
### 1. 机器学习（machine-learning/）
- 技术栈：Python、Pandas、Scikit-learn、Matplotlib
- 核心内容：回归预测、分类建模、探索性数据分析


### 2. Python编程（python-projects/）
- 爬虫：爬取新闻、壁纸、招聘数据并导出Excel
- 自动化：浏览器自动化、Excel表格生成
- 数据分析：电商客户分析、天气预报查询系统


### 3. Web全栈开发（web-development/）
- 后端：Flask搭建接口服务，封装MySQL数据库操作
- 前端实战：企业网站登录页、样式开发
- 前端基础：作业提交检查功能、UI组件（密码框）、创客企业网站页面开发
